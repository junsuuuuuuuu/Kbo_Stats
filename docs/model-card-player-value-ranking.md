# AI 선수 가치 랭킹 모델 카드

## 목적

단일 OPS·ERA 정렬이 아닌 공격, 장타, 출루, 주루, 제구, 꾸준함, 출장량, 나이와 팀 내
기여 비중을 결합해 시즌별 0~100 가치 점수를 제공한다. 실제 WAR 데이터가 없으므로
WAR로 표현하지 않으며, 설명 가능한 데이터 기반 composite score로 정의한다.

## 계산 방법

- 같은 시즌의 유효 후보 안에서 지표를 백분위로 변환한다.
- 타자 100타석, 투수 30이닝 이상을 기본 표본으로 사용한다.
- 최근 최대 3개 유효 시즌의 주 지표 표준편차가 작을수록 꾸준함 점수가 높다.
- 팀 기여는 득점·타점·홈런 또는 승·세이브·홀드·탈삼진의 팀 내 비중으로 계산한다.
- 역할별 가중합을 0~100 AI Score로 반환한다.
- 세부 점수가 높은 상위 3개 영역을 랭킹 이유로 제공한다.

점수는 시즌 내 상대 평가이므로 서로 다른 시즌의 절대 우열을 직접 의미하지 않는다.

## 출력

- 시즌 전체 순위와 팀 내 순위
- AI Score와 역할별 세부 점수
- TOP3 가치 근거
- 시즌·팀 필터

## 한계

- 수비 세부 기록, 구장 효과, 연봉과 공식 WAR가 없어 실제 시장가치와 다르다.
- 구원투수와 선발투수의 역할 차이는 세이브·홀드·이닝 점수로 일부만 반영한다.
- 가중치는 제품 정책이며 추후 사용자 프리셋이나 learning-to-rank label이 생기면 교체한다.
