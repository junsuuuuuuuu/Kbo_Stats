# 데이터 출처 및 사용 주의사항

## 범위

이 프로젝트는 사용자가 제공한 `batting_stats.csv`, `pitching_stats.csv` 두 파일만 원천
데이터로 사용한다. 선수 식별에는 원본의 KBO `player_id`를 사용하며, `source_url`은 원본
선수 페이지를 추적하기 위한 provenance 필드로 보존한다.

## 저장소 정책

- `data/raw` 원본 CSV는 `.gitignore`와 `.dockerignore`로 공개 저장소 및 이미지에서 제외한다.
- `data/processed`에는 전처리된 통계가 있으므로 공개 전 원데이터 제공처의 이용약관과
  재배포 허용 범위를 별도로 확인해야 한다.
- 모델 artifact와 예측 결과도 원천 통계에서 파생된 데이터이므로 동일한 확인이 필요하다.
- 이 프로젝트는 KBO 또는 구단의 공식 서비스가 아니며, 통계의 공식성을 보증하지 않는다.

## 공개 전 확인 항목

1. CSV를 수집·제공한 사이트와 수집 일자를 README에 기록한다.
2. 원본 및 가공 통계의 저장·가공·공개·상업적 이용 허용 범위를 확인한다.
3. 허용 범위가 불명확하면 공개 저장소에는 코드와 작은 예제 데이터만 두고 전체 데이터는
   사용자가 직접 준비하도록 안내한다.
4. 선수 평가 결과가 사실 단정이 아닌 통계 모델의 추정치임을 UI와 모델 카드에 표시한다.

현재 저장소만으로는 원본 제공처의 구체적인 라이선스를 확정할 수 없다. 따라서 이 문서는
사용 허가를 부여하지 않으며, 실제 공개·배포 전 별도 확인이 필요하다.
